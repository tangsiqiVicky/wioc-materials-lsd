## 开发
> 无法正常预览项目效果时，请先检查是否正常安装依赖，再查看启动服务是否存在报错 使用内部组件库需要重新配置

```bash
# 第一步
npm config set registry=http://registry.npmjs.org

# 第二步
npm install

# 启动服务
npm run dev
```




#上传组件包

```bash
# 第一步 配置地址
npm config set registry=http://registry.npmjs.org

# 第二步 登陆账户 输入用户名、密码、邮箱
npm login

# 第三步 上传 需要更新package.json version版本号
npm publish


```
